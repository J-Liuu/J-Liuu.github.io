Cooking Havoc RPG

Duration: Development Project

Overview

A static cooking/exploration RPG that combines cooking mechanics with data-driven gameplay. Features a backend system for tracking player behavior and game analytics.

My Contribution

Role: Systems Programmer & Data Architecture

I built a comprehensive Twine-based data-logging system that tracks player location visit counts and sends them in real-time to Google Sheets. Utilized Harlowe scripting language and integrated calls to Google Apps Script web app for seamless data transmission. The system was inspired by Blizzard's Datahooks architecture used in their MMORPG systems, allowing for detailed player behavior analysis and game balancing insights.

p5.js Flower - Procedural Graphics

Duration: 2 Weeks

Overview

This project was inspired by J√°nos Karsai and explores procedural flower generation using mathematical principles. The flower is generated in 3D using trigonometric functions and parametric equations.

My Contribution

I developed this interactive flower visualization applying various mathematical concepts including:

  • Polar Coordinates: Used to generate the petal formations.
  • Power Functions: Modify petal sharpness and curvature.
  • Exponential Decay: Shapes the stem and gradual cone-like structure.
  • Bump Mapping: Adds texture variations to the petal edges.

The flower dynamically updates using p5.js, demonstrating mathematical visualization techniques.
