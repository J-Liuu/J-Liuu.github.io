p5.js Flower - Procedural Graphics

Duration: 2 Weeks (Graphics Programming Course)

This project explores procedural graphics generation using mathematical principles. The flower is generated in 3D using trigonometric functions and parametric equations. Concepts applied include:

  • Polar Coordinates: Used to generate the petal formations.
  • Power Functions: Modify petal sharpness and curvature.
  • Exponential Decay: Shapes the stem and gradual cone-like structure.
  • Bump Mapping: Adds texture variations to the petal edges.
  • Orbit Control: Allows interactive rotation and zooming in WEBGL mode.

The flower dynamically updates using p5.js, demonstrating mathematical visualization techniques.
